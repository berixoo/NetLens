# NetLens — 局域网代理服务暴露检测工具

基于 PySide6 的桌面应用，用于在局域网和校园网环境中检测暴露的代理服务。

## 功能特性

- **两阶段校园网扫描** — 先快速发现存活主机，再深度扫描代理端口
- **多线程端口扫描** — 可配置线程池（最高 1024 线程）
- **协议自动识别** — HTTP CONNECT、SOCKS4、SOCKS5
- **代理连通性验证** — 通过代理实际转发流量，确认可用性
- **系统代理切换** — 一键将检测到的代理应用到 Windows 系统设置，或关闭系统代理
- **代理记忆** — 自动保存已验证的代理记录，下次可直接选用，无需重新扫描
- **风险等级评估** — CRITICAL / HIGH / MEDIUM / LOW / INFO 五级分类
- **多种目标格式** — 单 IP、CIDR、短横线范围、逗号分隔列表、文件导入
- **扫描模式** — 本机子网 (/24)、校园网段 (/16)、自定义范围
- **导出报告** — CSV、JSON、纯文本日志
- **暗色主题 UI** — Catppuccin Mocha 配色

## 扫描模式

| 模式 | 范围 | 适用场景 |
|------|------|----------|
| Local Subnet | /24（254 台主机） | 当前所在子网 |
| Campus Segment | /16（65534 台主机） | 整栋楼/整个校区网段 |
| Custom | 用户自定义 | 任意 IP 列表、CIDR 或范围 |

超过 256 台主机时自动启用两阶段扫描：
1. **发现阶段** — 用 5 个常用代理端口（7890、1080、8080、443、80）快速探测，超时 1 秒
2. **深度扫描** — 仅对存活主机进行全端口扫描 + 协议识别

## 默认代理端口

| 端口 | 常见服务 |
|------|----------|
| 7890 | Clash HTTP 代理 |
| 7891 | Clash SOCKS5 代理 |
| 1080 | 通用 SOCKS 代理 |
| 10808 | V2Ray / Xray HTTP 代理 |
| 10809 | V2Ray / Xray SOCKS5 代理 |
| 8080 | 通用 HTTP 代理 |
| 8118 | Privoxy |
| 3128 | Squid |

## 安装与运行

```bash
pip install -r requirements.txt
python app.py
```

## 风险等级说明

| 等级 | 含义 |
|------|------|
| CRITICAL | 开放无认证代理，已验证可达 — 立即风险 |
| HIGH | 检测到开放无认证代理 — 高风险 |
| MEDIUM | 需认证的代理或无法识别的服务 — 中等风险 |
| LOW | 开放端口但非代理服务 — 低风险 |
| INFO | 端口关闭或不可达 — 仅供参考 |

## 界面操作

1. **选择扫描模式** — 下拉框选择 本机子网 / 校园网段 / 自定义
2. **点击校园扫描** — 一键扫描校园网段，自动开启两阶段模式
3. **检测到代理后弹窗** — 可选择「使用此代理」直接设为系统代理，或「跳过并继续扫描」
4. **结果表选中某行** — 点击「应用到系统」手动应用，或「关闭系统代理」关闭
5. **已保存代理** — 点击「已保存代理」查看历史记录，一键应用或删除
6. **导出** — 点击导出 CSV / JSON / 日志保存扫描结果

## 代理记忆

扫描发现的代理会自动保存到 `proxy_memory.json`，记录包括：
- 地址、类型、延迟、是否需要认证
- 首次发现时间、最后发现时间
- 使用次数、验证成功次数

下次启动时，点击「已保存代理」可直接查看和使用之前保存的代理，无需重新扫描。
